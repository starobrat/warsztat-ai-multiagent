# Rozwiązania

Kompletne, działające wersje ćwiczeń. Struktura lustrzana do katalogów ćwiczeń.

W trakcie szkolenia: zerknij tu, jeśli utkniesz - ale najpierw spróbuj sam.
Rozwiązania korzystają z tych samych klocków co ćwiczenia (`common/`, `common/tools/`).

- `solutions/ex_01_simple_call/solution.py` - wypełniony prompt + parametry
- `solutions/ex_02_function_calling/solution.py`
- `solutions/ex_03_agentic_loop/solution.py`
- `solutions/ex_04_hello/agent.py` - napisana instrukcja pierwszego agenta
- `solutions/ex_05_sql_agent/agent.py` - wypełniona instrukcja + narzędzia
- `solutions/ex_07_sql_agent_tuning/agent.py` - poprawiona instrukcja (eval na zielono)
- `solutions/ex_08_report_system/agent.py` - wypełniony planner i report_writer
- `solutions/ex_09_tests/test_report_system.py` - podłączony test systemu raportowego
- `solutions/ex_10_guardrails/agent.py` - napisany callback before_tool

Bez osobnego rozwiązania (z założenia):
- `ex_00_setup` - smoke test i schemat, nie ma kodu do napisania.
- `ex_06_evaluation` - kodu nie piszesz, nagrywasz własne case'y eval w `adk web`.
- `ex_09_tests/test_sql_agent.py` - jest gotowy w samym ćwiczeniu jako wzór.
