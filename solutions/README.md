# Rozwiązania

Kompletne, działające wersje ćwiczeń. Struktura lustrzana do katalogów ćwiczeń.

W trakcie szkolenia: zerknij tu, jeśli utkniesz - ale najpierw spróbuj sam.
Rozwiązania korzystają z tych samych klocków co ćwiczenia (`common/`, `common/tools/`).

- `solutions/01_simple_call/solution.py` - wypełniony prompt + parametry
- `solutions/02_function_calling/solution.py`
- `solutions/03_agentic_loop/solution.py`
- `solutions/05_sql_agent/agent.py` - wypełniona instrukcja + narzędzia
- `solutions/07_sql_agent_tuning/agent.py` - poprawiona instrukcja (eval na zielono)
- `solutions/08_report_system/agent.py` - wypełniony planner i report_writer
- `solutions/09_tests/test_report_system.py` - podłączony test systemu raportowego

Bez osobnego rozwiązania (z założenia):
- `00_setup`, `06_evaluation` - nie ma kodu do napisania (smoke test, schemat, evalset).
- `04_hello`, `10_guardrails` - to gotowe agenty-referencje; ich `agent.py` JEST wzorcem.
- `09_tests/test_sql_agent.py` - jest gotowy w samym ćwiczeniu.
